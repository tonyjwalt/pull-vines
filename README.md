pull-vines
==========

Pulls vines from a twitter feed and places them in a list.

### Notes
The keys in "get-tweets.php" need to be changed to your keys:
* $consumerkey = "123456";
* $consumersecret = "123456";
* $accesstoken = "123456";
* $accesstokensecret = "123456";
